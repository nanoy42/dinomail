DinoMail

A simple mail management web interface

GitHub Documentation Try live

Functionalities

Domains, Users and Aliases

You can easily manage the virtual domains, users and aliases. There are stored in database and can be used with postfix by instance.

Check your domains

DinoMail checks for DKIM, SPF and DMARC records on your virtual domains.

Responsive

The website renders nicely on all screens. For android smartphones, an app is also available on the Google Play Store.

API

Not happy with the existing clients or need for automation ? You can do pretty much every action using the API.

Mobile app for android

There is a smartphone app using DinoMail API. You just have to connect using a correct account and you can manage the domains, the users and the aliases.

More information

Join the team

You want to help improving DinoMail. Don't hesitate to go on GitHub and to open an issue or help to close one.

Try it live

Try it live is a live version of DinoMail that is not connected to any mail server. The database is flushed every 24 hours (at midnight). The login and password are admin/admin.
